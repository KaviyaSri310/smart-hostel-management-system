# **App Name**: SmartHostel

## Core Features:

- Login & Registration: Secure login and registration process for students and admins.
- Group Code Generation: Generate a unique group code for students to invite roommates and manage roommate TTL.
- Room Selection & Booking: Interactive room selection with real-time availability and bed layout display. Manage status updates after requests are made
- Student Dashboard: Personalized dashboard for students to view room details, request repairs, and submit leave requests.
- Admin Dashboard: Admin dashboard for managing room approvals, leave requests, and repair requests.
- Repair Request Tool: Submit repair requests with categorization and issue description. The LLM reasons whether the selected issue requires images as an addition to the explanation
- Technician Heatmap Visualization: Visual representation of repair request density by room/floor for admin insights.

## Style Guidelines:

- Primary color: Teal (#008080) for a modern and calming feel.
- Background color: Light gray (#D3D3D3) for a clean and spacious layout.
- Accent color: Navy (#000080) to complement the teal and provide contrast.
- Body and headline font: 'Poppins' (sans-serif) for clear, contemporary text.
- Flat style icons related to hostel management: bed, wrench, calendar, key.
- Modern flat UI with hostel elements with rounded cards and modals with soft shadows and 12px corner radius.
- Smooth page transitions for an engaging user experience.
